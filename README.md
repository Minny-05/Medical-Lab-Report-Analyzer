# Medical-Lab-Report-Analyzer
AI Medical Lab Report Analyzer converts complex lab reports into simple explanations. It extracts test values, compares them with medical standards, and uses AI to generate safe, patient-friendly summaries without providing diagnoses.
# Lab Report Intelligence Agent

AI-powered system to simplify medical lab reports into patient-friendly explanations.

# Features
- PDF lab report upload
- OCR support (scanned reports)
- Benchmark comparison
- AI-generated explanation (Groq LLM)
- Family profile management
- Doctor & specialist details

# Architecture
Frontend: React + Tailwind  
Backend: Flask + OCR + RAG + Groq
